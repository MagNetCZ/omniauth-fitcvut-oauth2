# OmniAuth CTU FIT

This gem contains the ÄŒVUT/CTU FIT OAuth 2.0 strategy for OmniAuth 1.0.

## Installing

Add to your `Gemfile`:

```ruby
gem 'omniauth-fitcvut-oauth2', '~> 0.2.1'
```

Then `bundle install`.

## Basic Usage

    use OmniAuth::Builder do
    	provider "fitcvut_oauth2", ENV['FIT_APP_ID'], ENV['FIT_APP_SECRET']
    end
